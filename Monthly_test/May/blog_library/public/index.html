Blog Library

Welcome to Blog Library

Please use the following routes :

(POST) /blogs/add --> to add new blog

(GET) /blogs/all --> to get all blogs

(PUT) /update/:id --> to update a blog according to id

(DELETE) /delete/:id --> to delete a blog according to id

(GET) /gettitle/:name --> to get all blogs in a sorted way according to Blog_Title

(GET) /getauthor/:name --> to get all blogs in a sorted way according to Author
